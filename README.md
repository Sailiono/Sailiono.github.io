# Clark Cui — Portfolio

Source for [sailiono.github.io](https://sailiono.github.io), a dependency-free bilingual portfolio hosted with GitHub Pages.

## Local preview

```bash
python3 -m http.server 8765
```

Open <http://localhost:8765>.

## Structure

- `index.html` — accessible bilingual portfolio content;
- `styles.css` — responsive visual system with no external assets;
- `script.js` — English/Chinese language preference;
- `404.html` — GitHub Pages fallback;
- `robots.txt` and `sitemap.xml` — search-engine discovery.
